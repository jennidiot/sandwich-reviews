GitHub Sync

Connect to a shared repository

One-time setup (≈5 minutes)
1. Create a new GitHub repo — e.g. sandwich-data. It can be private.
2. Go to GitHub → Settings → Developer settings → Tokens (classic)
3. Generate a new token with the repo scope ticked. Copy it.
4. Fill in the fields below and click Save & Connect.
5. Share the same token + repo details with everyone who needs write access.
The app will create and update reviews.json in your repo automatically.

⚠ The token is stored in your browser's localStorage. Avoid using this on shared or public computers.
